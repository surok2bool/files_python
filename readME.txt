Учебный проект.
Задача:
Создать приложение для работы с файлами.
	1) чтение файлов
	2) получение информации о файлах (размер, дата изменения и т.д.)
	3) вывод полученной информацйии на экран
	4) запись полученной информации на экран
	5) сравнение текущего состояния файлов с прошлыми результатами работы приложения
